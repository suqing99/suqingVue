# suqing-practice

## 简介

该repo是用来练手的一个小项目，用来练习Vue2.x, Vue3.0, TypeScript, Unit Test等。

## 使用方法

```bash
  git clone https://github.com/lovelyApe/suqing-practice.git
  cd ./suqing-practice
  
  npm install
  npm run serve
```
